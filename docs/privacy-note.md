# Privacy Note

This portfolio uses lightweight interaction analytics for product-quality improvements.

## What is tracked

- Project interaction events
- Resume click events
- Contact action events
- Basic media/network failure events

## What is not collected

- No account data
- No payment data
- No sensitive personal profile enrichment

## Purpose

Analytics events are used only to understand UX quality, improve conversion paths, and identify broken flows.

## Retention and minimization

Event payloads should remain minimal and avoid personally sensitive content.

## Contact

For privacy concerns, contact the site owner through the public contact channels listed on the portfolio.
